## О боте
Простой телеграм бот для конвертации **нескольких** веб-страниц в формат PDF. Пользователь может выбрать скачать файл в обычном размере или с учелечнным для мобильных устройств.

## ✨ Как бот работает:
1. Пользователь кидает текст, содержашие несколько ссылок (в текстовом виде или в виде гиперссылок)
2. Бот конвертирует в pdf каждую ссылку по порядку и сохраняет всё в один архив
3. Отправляется пользователю zip-архив со всеми pdf файлами (Если ссылка одна, то отправляетс pdf-файл)

## ⚙️ Установка и настройка

**Примечание:** Рекомендуется запускать бота на хостинге или сервере, любой телеграм бот требует постоянно запущенного кода для постоянной работы.

1. Предварительные требования
   - Python 3.8+
   - Токен вашего телеграм бота ([Где получить](https://t.me/BotFather))
   - Установленный [wkhtmltopdf](https://wkhtmltopdf.org/downloads.html)
  
2. Клонирование репозитория
   <br>`git clone https://github.com/N0Fanru/URL-to-pdf-telebot`
   <br>*или*
   <br>скачать как ZIP файл и распаковать

3. Установка зависимостей
   <br>В командной строке: `pip install -r requirements.txt`

4. Создайте файл `.env` на основе файла `.env.example`

5. Запустите бота
   <br>`python bot.py`
